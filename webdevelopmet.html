Web Development

Many are again confused with web design and development. Web design is what the website will look like. Which is usually made with a design tool, such as Photoshop or Illustrator. Development is the realization of the code after seeing the design.

The front end is the part of a web site we see in the browser. This is also called client side. HTML, CSS and JavaScript are used to create the visual end of this front end or web site. All web sites need HTML and CSS. The contents of the web site are displayed with HTML. And what the content will look like, which one will be on the side, etc. is fixed with CSS. Javascript is used to perform other functional tasks including animation on the client side.

Sites that do not change the content of the site very much, with only a few pages, those sites can usually be created with HTML, CSS and JavaScript.

Think of thousands of pages on a web site. Once again the contents of all the pages need to be updated regularly, then it will take a long time to create a web site with simple HTML, CSS and JavaScript. It will take a long time to update each and every time. The process of updating is boring. Programmers don't work boring. Web programming languages ​​are used to solve this problem. And these languages ​​run on the server. The server side part is called backend.

Although only HTML, CSS and JavaScript are used for frontend development, there are many programming languages for backend development. The top few web programming languages are PHP, ASP, Java, JavaScript, Ruby, Python and so on. You might be surprised to see the name of JavaScript in the backend programming list. With javascript, the server side code can be written on the front end as well.

Another part of the backend is the database. Database is used to organize web site data. Some of these database management systems are Oracle, MySQL, Microsoft SQL, and so on. SQL is used to view data from these database management systems, store data in a database, update data and delete data. SQL means Structured Query Language.

If someone develops only the front end part of a web site, they are called front end developers. If someone develops a backend part, they are called backend developers. If you want to develop the frontend and backend components together, then tell you fullstack web developer.
